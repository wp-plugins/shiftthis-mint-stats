View your Mint statistics from within the WordPress Admin.

== Description ==

The WordPress Mint Stats Plugin will allow you to view your Mint statistics from within the WordPress Admin area by embedding an iframe within a new Admin Page. It also includes the ability to include the Mint tracking javascript into your WordPress Theme.

== Installation ==


1. Upload 'mint_stats.php' to the 'wp-content\plugins' directory
2. Activate the plugin titled 'ShiftThis | Mint Stats' through the 'Plugins' menu in WordPress
3. Access the **Options** Tab in your Admin Panel and select the **Mint Stats** SubMenu Item.

== Mint Options ==

* **Mint Admin Page** - Enable the Mint Tab in the WordPress Admin Menu.
* **Mint Menu Title** - Set the title of your Mint Tab.
* **Mint User Level** - Set the minimum User Role for who can access the Mint Admin Menu Page.
* **Mint Logging Status** - Enable the javascript to allow Mint Tracking.
* **Mint Directory** - Enter the directory Mint is installed into.
* **Script Location** - Enter the location of the Mint Javascript.
* **Mint iFrame Height** - Enter the height (in pixels) you prefer the Mint iFrame to display on the Mint Tab.
